Keyboard Shortcuts

Key Description
a recentre the view to the object
t manually flush the texture cache (works even if the timed texture flush is enabled)
l held down to change the angle of the direct light with the left mouse button
shift-l toggles the lights
up arrow increase the mip map base level
down arrow decrease the mip map base level
m toggles the max mip map level to be the same as the base one, effectively disabling mip mapping (also available on the toolbar)
left arrow manually lowers the detail level
right arrow manually raises the detail level
d toggles the detail level control from manual to the standard TGE pixel size determination (also available on the toolbar)
page up changes the display to the previously loaded object
page down changes the display to the next loaded object
h toggle the HUD (also done through the Display Properties window General tab)
n toggles the display of nodes and their links (also done through the Display Properties window Render tab and the toolbar "++" icon)
w toggle the window to always be on top of other application windows
g toggles the display of the grid
1 change the object view to orthographic, top down
2 change the object view to orthographic, bottom up
3 change the object view to orthographic, front facing
4 change the object view to orthographic, back facing
5 change the object view to orthographic, left facing
6 change the object view to orthographic, right facing
7 change the object view to perspective, free camera
F1 - F8 activate a user defined preset (defined from the preset drop-down menu)
